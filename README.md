# LightLock - Ambient Light Jump Detector for macOS

LightLock - это утилита для macOS, которая читает данные с датчика
освещённости (ALS) и обнаруживает резкие изменения уровня света.\
Подходит для автоматического гашения экрана или анализа освещённости в
реальном времени.

## Возможности

-   Сбор показаний датчика ALS на MacBook / iMac.
-   Выявление резких скачков по производной.
-   (Опционально) мгновенная блокировка экрана через
    `pmset displaysleepnow`.
-   График освещённости в реальном времени (режим DEBUG).
-   Режим без графики для фоновой работы.

## Требования

-   macOS с аппаратным датчиком освещённости.
-   Установленный Python 3.
-   Пакеты `macals`, `matplotlib`.

## Установка зависимостей

``` bash
python3 -m pip install -r requirements.txt
```

## Запуск

``` bash
python3 light_lock.py
```

## Структура репозитория
